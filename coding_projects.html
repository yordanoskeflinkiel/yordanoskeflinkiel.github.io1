This project replicates a hotel booking site. Login and registration are implemented via PHP.
Upon logging in, users are directed to the main page where they can explore different room styles, accommodations offered, and pricing details.
When ready to book, users proceed to the booking page to select an available date, room amount, style, and the number of guests.
After booking, the details are accessible on their profile page.
Current limitations include authentication for payments and emails, as well as syncing between instagram social media occasonally going down.
